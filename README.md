# 🌱 InAmigos Foundation - NGO Awareness Website

A modern, responsive NGO awareness website developed using HTML, CSS, and JavaScript as part of the AI Web Development Internship Task.

## 📌 Project Overview

This website is designed to spread awareness about the initiatives and social impact of the InAmigos Foundation. It provides information about the NGO, its ongoing projects, community impact, gallery, and encourages visitors to volunteer or support the organization.

## ✨ Features

- Responsive Navigation Bar
- Full-screen Hero Section
- About InAmigos Foundation
- Ongoing Projects Section
- Social Impact Statistics
- Image Gallery
- Call-to-Action Section
- Footer with Social Media Icons
- Scroll Animations (AOS)
- Mobile Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- AOS (Animate On Scroll)
